# Voice Memo Uploader

A simple Android application for recording and uploading voice memos.

## Latest Build

📥 **[Download Latest APK](https://github.com/clawthor0/voice-memo-uploader/releases)**

## Features

- Record voice memos quickly
- Upload directly to cloud storage
- Simple, intuitive interface

## Getting Started

### Development

This repository uses a structured branching strategy:

- **`dev`** - Active development branch
- **`master`** - Production releases

See [BRANCHING.md](BRANCHING.md) for detailed workflow information.

### Building Locally

```bash
./gradlew assembleRelease
```

The APK will be available at `app/build/outputs/apk/release/app-release.apk`

## Releases

All releases are automatically built and published when code is merged to the `master` branch. 

[View all releases](https://github.com/clawthor0/voice-memo-uploader/releases)

## Contributing

1. Create a feature branch from `dev`
2. Make your changes
3. Submit a PR to `dev` for review
4. After approval and testing, create a PR to `master` for release

See [BRANCHING.md](BRANCHING.md) for detailed guidelines.
